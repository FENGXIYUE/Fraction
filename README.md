# Fraction
第三次测试:分数处理工具
设计一个表示分数的类Fraction。这个类用两个int类型的变量分别表示分子和分母。

1.分数转小数
2.两个分数之间的四则运算，将自己以“分子/分母”的形式输出到标准输出，并带有回车换行。如果分数是1/1，应该输出1。当分子大于分母时，不需要提出整数部分，即31/30是一个正确的输出。

注意，在创建和做完运算后应该化简分数为最简形式。如2/4应该被化简为1/2。
